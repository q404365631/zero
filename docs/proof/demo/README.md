# ZERO Demo Proof Pack

This directory is generated by `scripts/proof_pack.py`.

It is a deterministic paper-mode proof pack for repository launch readiness.
It is not a paper-vs-live P&L correlation claim and does not imply live trading
profitability.

Verify it with:

```bash
PYTHONPATH="$PWD/engine/src" scripts/proof_pack.py --check
```
